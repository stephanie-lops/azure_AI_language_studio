# Language Studio no Azure AI
Bootcamp Back-end Avanade

Este repositório contém um resumo dos principais conceitos estudados sobre **Azure AI Language Studio**, abordando análise de texto, bots, compreensão de linguagem e processamento de fala.


## 1. Análise de Texto (Text Analytics)

A Análise de Texto no Azure AI permite extrair informações valiosas de textos de forma automática. Com ela é possível:

- Identificar sentimentos (positivo, negativo ou neutro)
- Detectar entidades (pessoas, empresas, locais, datas)
- Extrair frases-chave
- Reconhecer idiomas
- Classificar textos por categoria  

Muito usada em atendimento ao cliente, análise de redes sociais e análise de feedbacks.


## 2. Bot no Azure (Azure Bot Service)

O Azure Bot Service permite criar chatbots inteligentes integrados com serviços de IA.

Principais funcionalidades:
- Criação de bots para WhatsApp, Teams, sites e aplicativos
- Integração com Language Studio, QnA e CLU
- Automação de atendimento, suporte e vendas
- Uso de respostas automatizadas baseadas em IA

Ferramenta essencial para chatbots corporativos e assistentes virtuais.


## 3. Compreensão de Linguagem (Natural Language Understanding – NLU)

Esse recurso permite que sistemas entendam a intenção do usuário, indo além de palavras-chave.

Principais usos:
- Identificação de intenções (comprar, cancelar, solicitar)
- Detecção de entidades no texto
- Criação de modelos de linguagem personalizados
- Integração com bots e automações

Base da IA conversacional moderna.


## 4. Estúdio de Fala (Speech Studio)

O Speech Studio trabalha com voz e áudio, permitindo transformar fala em texto e texto em fala.

Principais recursos:
- Speech-to-Text (fala para texto)
- Text-to-Speech (texto para voz)
- Tradução de fala em tempo real
- Reconhecimento de voz por locutor

Muito usado em acessibilidade, call centers, dublagens, assistentes virtuais e aplicativos mobile.


## 5. Language Studio (Azure AI Language)

O Language Studio é a interface visual da Microsoft Azure para testar, treinar e usar modelos de linguagem sem necessidade de programação.

Com ele é possível:
- Testar análise de sentimentos
- Criar modelos de classificação de texto
- Criar modelos de extração de entidades
- Treinar compreensão de linguagem
- Avaliar resultados em tempo real

Funciona como um laboratório prático de IA aplicada à linguagem.


## Tecnologias Utilizadas

- Microsoft Azure
- Azure AI Services
- Azure Language Studio
- Azure Bot Service
- Speech Studio
